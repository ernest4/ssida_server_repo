How SIDDA Data Processing Server Works?

by Ernestas M. - SIDDA Team.
Oct 18, 2018 | 1 min read

Users share their movement data from Flow [working title] app with this server. This can be seen on Live Data

The server stores the data in an AWS RDS PostgreSQL database.

The information is then retreived by this server for display and/or running analytics to create a heat map. You can explore this data in Stored Data

The heat map is then sent back to the users' devices so they can avail of unbiased, scientific opinions of the quality of the environment they find themselves in.
